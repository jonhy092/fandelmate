PROYECTO TITULADO FAN DEL MATE.
Tal proyecto está en construcción. Fan del mate será un sistema donde se podrá acceder solo como visualización, como comprador/usuario y como administrador, ambos pueden loguearse.
El administrador es quien podrá realizar modificaciones correspondientes al stock, precios o añadir/quitar distintos productos sobre su modelo de negocios.
Tal proyecto requerirá del uso de API REST, con sus detalladas funcionalidades.
